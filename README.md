## This is a real-time SPA forum

This app uses Laravel 7.2 and Vue.js 2.5

- SPA
- SASS environment 
- CSS: vuetify
- vuex, vue-router
- Database is MySQL
- XAMPP
- Auth implemented using Sanctum
- Notification using Pusher
